Shortcut

Shortcut links for the Generic Profile Configuration and Updater are created at Installation. They are typically found on the Windows Start menu.

An option to create a Shortcut link to a new Profile Configuration is found on the Configure New Pictures Screen.

Shortcuts will only be created in the MyReels group in the Start menu Programs Area if it exists, and the new Shortcut name will be the current Profile Configuration directory name. MyReels never deletes Shortcuts, but no Profile Configuration directory will ever have more than one Shortcut unless a Profile Configuration is overwritten. Greater functionality should come with newer Versions.
