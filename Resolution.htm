Resolution

This game is designed for a Screen Resolution of 800 X 600 pixels and will resize for 1024 X 768, 1152 X 864, 1280 X 1024 or wide screen 1280 X 720, 1366 X 768, 1600 X 900. In any higher resolutions some fonts may clip and scaling issues can occur in any situation- for instance the Game Screen runs off the bottom edge 1280X1024 on a certain XP machine with an S3 graphics chip and CRT monitor. A program like PrgLnch can launch MyReels in a lower resolution, but being a VB6 compilation MyReels can only be run from the Primary Display in a multi-monitor environment.

Choosing the Big Pictures option in any higher resolution than 800X600 won't improve the quality per-pixel of a rendered Picture in-game.

For Pre-Vista Users, setting Text/Fonts to anything exceeding 100 % of "normal" size through Control Panel/Windows Display Properties/Settings may present issues with the display of the Game Fonts.

Please Quit the game before changing the Resolution or other settings of your Windows Display.
