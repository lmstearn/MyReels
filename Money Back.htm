Money Back

The Money Back Feature is a No Win Bonus scheme where a Player can use pre-formulated strategies to maximise returns. The basic idea is that the Player is refunded all bet monies (Money Back Pot) if there has been no win in a specified number of turns. The Feature is triggered when a win greater than or equal to a specified amount has occurred and is configurable from the Profile Configuration Manager Screen.

The number of different betting Strategies for the Feature can be viewed in the About Screen where the Optimal Money Back Strategy can be generated.
