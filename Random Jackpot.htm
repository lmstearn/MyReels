Random Jackpot

A Random Jackpot Feature is configurable from the Profile Configuration Manager Screen.The % add on to the Value Of Game is exactly 5%. For a Jackpot of 1 calculate a random between 1 & 20. For a Jackpot of 1.1 calculate a random between 1 & 22. For a Jackpot of J calculate a random between 1 & 20 * J. So the 5% Value Of Game is calculated as 1 / (20 * J) * (20 * J) / 20 = 1/20 or 5%.

In practice a "Secret" value is calculated & stored on Game generation. At each Spin the value J is incremented and a new Random number is generated in sequence. If it is equivalent to the "Secret" number, the Jackpot J is won.

It should be noted that if a Jackpot value of 345.5 is won, for example, another random test is performed to determine whether 345 or 346 should be payed. 345.4 is rounded down to 345 and 345.6. is rounded up to 346.

Disabling and then re-enabling the Feature will reset the Jackpot values to those displayed on the Profile Configuration Manager Screen.
