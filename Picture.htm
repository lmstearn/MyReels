Picture

Every Picture in the current game is visible on the Profile Configuration Manager Screen, the Edit Reels Configuration Screen and on at least one of the 5 Reels of the Game Screen. Can be selected from files of ICO, WMF, BMP, JPG or GIF extensions, which MyReels converts to 43k bitmaps. There can be any from 3 to 14 different Pictures specified in a Profile Configuration and a new set of Pictures can be configured from the Configure New Pictures Screen.

MyReels uses the PaintPicture method which in turn uses the BitBlt function to size and stretch pictures on the various screens.
