Updater

In Version 3.0 and later, a more convenient way of updating MyReels is introduced with the Updater. At Installation the file is copied to the MyReels base directory and a Shortcut is created on the start menu.

The Updater extracts updated application files to the MyReels directory each run, but the updated script itself will not be executed until the next time the Updater runs.

If the Generic Quotes are to be updated, the new maximum size of the Database is read only at the beginning of a game despite subsequent modifications, so slotdata.s$t is included in the package.

The Updater cannot be used to update older Versions than 3.X.
