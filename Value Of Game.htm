Value Of Game

If the Monte Carlo option is not used, the Value of Game (VOG) is an expectation calculated for games that use no Substituters, Free Games or Free Spins. It will always be incorrect and overestimated when Scatters are configured as well, because some Scatter combinations are summed twice. The problem of sampling Scatters and getting a reasonable estimate for hypothetical distributions unconditional on Reels arrangements is then best left for future Versions.

The VOG can be calculated for Pictures with Free Game Options. The problem of Free Spins is a bigger challenge as are Substituters, so will probably not be completed in the foreseeable future.

5% is then added to VOG if the Random Jackpot is selected and the Probability Of No Prizes component is estimated using the current Random Jackpot value.

None of the above complications arise when the Monte Carlo option is used.

The Random Jackpot option, if selected, will be automatically incorporated in the Monte Carlo simulation.

The resultant value can then be used with the Probability Of No Prizes and the Optimal Money Back Strategy to determine the final VOG. This Money Back component estimate of VOG is based on steady state comparison ratios.

The VOG is displayed in the Profile Configuration Manager Screen and the About Screen.
