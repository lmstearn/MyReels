Some Improvements are ever abounding for this type of game:

  • Vary the number of Reels.
  • Increase the number of Paylines.
  • Allow for Picture animation.
  • Allow Pictures to move sideways, diagonally or ... in a helical fashion?
  • Allow Pictures shapes other than square.
  • Improve graphical environment e.g. DIRECTX (tm) & Alternatives.
  • Allow effects in animation- 3D?.
  • Allow Player to move controls around the Screen to enhance the customizing experience.
  • Allow an animated "Banker" character organise loans & accounts.
  • Design a browser version of the game.
  • Multiplayer game over network featuring Multiplayer Random Jackpot.
  • Increase size of in-program directory names which means major overhaul of the old drive and file listboxes and SHGetPathFromIDList. Common dialog at least supports max size of 32k- and we also have Bigger Directories!
  • Allow a user selected subset of Quote Categories.
  • Allow different font styles and sizes to suit various categories of Quotes.
  • Refactor Quote Sounds so they can be used for Quote Thumbnails in Profile Configurations other than the Generic.
  • Introduce a themed approach to Sounds & Music config.
  • Allow a mini preview or a "what's next" of Quotes.
  • Play Sounds that match duration of Spin.
  • Play a tune or Sound for "near" misses.
  • Include Music from MP3 sources.
  • Expand the scope of all Picture Properties.
  • Improve image rendering after transform, (E. Lears drawings in particular)
  • Use a nudge option and allow Player the option to hold Reels in the Game Screen.
  • Introduce "Trails" or a "Snakes & Ladders" style narrative aspect to the game.
  • An option to automatically change Bet Multiplier to reflect corresponding Optimal Money Back Strategy.
  • Expand on the algorithms for the Value Of Game.
  • Develop a multilingual, locale/currency-aware MyReels.

Some of the above tasks are rather complex and would involve extensive program redesign. Surely there must be developers out there who would find some of these a challenge for their own programs. Any takers?
