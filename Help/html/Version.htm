Version

The current Version of this game is 3.2. A large number of bugs have been fixed from 1.0, along with some Future Improvements and an improved help file; details described in the Version History, More Version History and Even More Version History.

Version 2.X is compatible with Version 1.X Slotdata.s$t files but not 1.X Hallfame.s$t files. 1.X Hallfame.s$t files are unusable and should be discarded (but past glories not forgotten).

Any minor updates or post-release news items will appear on The Author's Homepage.

Versions 2.2X and later are compiled with VB6 and should work on any Win32 Platform.

Msflxgrd.ocx is not bundled with this version. Neither is favrites.txt for the Quotes at Installation, but a text file can be easily generated from the Configure Thumbnails Screen.
