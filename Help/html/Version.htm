Version

The current Version of this game is 3.3.1. A large number of bugs have been fixed from 1.0, along with Installation issues addressed, some Future Improvements, an Updater, and an improved help file. Ongoing version development is described in the Version History, More Version History and Even More Version History, and pre-release news of exciting new Versions will appear on The Author's Homepage.

Notes for MyReels Versions 1.X & 2.X:
Version 2.X is compatible with Version 1.X Slotdata.s$t files but not 1.X Hallfame.s$t files. 1.X Hallfame.s$t files are unusable and should be discarded (but past glories not forgotten).
The first releases of MyReels were made with the VB5 compiler, versions 2.2X and later are VB6, and as before, run on any machine where Win32 is defined.
The Database DLL this program initially used was DAO350.DLL. There are newer versions available in the form DA03**.DLL, and there presence in the system directory shouldn't adversely affect the Installation or performance of this program.
Msflxgrd.ocx was only bundled with versions up to 2.1X. Favrites.txt for the Quotes was also removed, but a text file can be easily generated from the Configure Thumbnails Screen.
