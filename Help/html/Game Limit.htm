Game Limit

The Player can specify a Limit or tolerance on the game on the General Tab of the General Options Screen. This value applies to all those displayed in the Game Statistics. Typically the Money On Hand is the quantity that first exceeds this Limit. Once this Limit is reached, the Player receives a TY message and must then commence a New Game.

The Game Limit cannot be reduced when at least one of the Statistics in the current game has already exceeded the new Limit.
