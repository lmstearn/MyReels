The About Screen consists of:

  • The top section contains the Version, "Mission" Declaration, Date, Time, UTC (Co-ordinated Universal Time - GMT for Brit. Empire fans) bias and the Hall_of_Fame Screen button.
  • The upper middle section displays various Statistics of various Prize Categories used in the game. If this Screen is invoked more than once during a game, the percentage rate of improvement from last invocation is shown. Of interest is the displayed proportion of Prizes won in each Category to the Sampled Value of Game. This value has a Tooltip that indicates the number of Spins and Effective Bet for Prizes won. If there have been no Prizes scored in a given Category then obviously no proportion can be displayed.
  • Above the Money Back and Random Jackpot Prize Categories is the Free Spin / Free Game No Win Bonus which may show a value and Tooltip if contributing to the Sampled Value of Game.
  • The lower middle section contains more general Statistics, here, the average bet size can be less than 1 if Free Games or Free Spins have been awarded.
  • The bottom section contains information regarding available Money Back Strategies. The Generate button will populate the list of Strategies when the Value Of Game is > 0 and < 100 percent. Select a value in the Bet Total to obtain the Optimal Money Back Strategy for that value. Notice the Value Of Game varying for different Bet Totals.
  • Below that is the "Back to Game" button returning to the Game Screen, buttons leading to the Device Info Screen and a System Information Facility, as well as a Statistics reset button. Fortunately resetting Statistics doesn't reset the Money On Hand value.
