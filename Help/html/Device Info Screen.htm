The Device Info Screen features:

Information the Windows GDI has stored about your computers graphics capabilities. It really doesn't have any direct bearing on the game except that if you are missing any of:

BitBlt
Capable of transferring bitmaps
Device Independent Bitmap
A BITMAPINFO structure describing the dimensions and colors of the bitmap, and an array of bytes defining the pixels of the bitmap.
Flood fill, StretchBlt & StretchDIB
Basic functions on bitmaps
there will be an impact on the GDI which will in turn affect all programs that process graphics. A typical Windows rig with printer attached will show the following section headers:

  • Raster Display.
  • Device Capabilities.
  • Raster Printer.
  • Device Capabilities.

As of Version 3.3.1, right clicking the OK button gets a list of data relating to the debugging of certain issues MyReels has with the graphical environment.
