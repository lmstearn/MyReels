The Configure Thumbnails Screen features:

  • On the left, a drive list, directory list, file list and a Picture file filter. A double click is required to change directories, a single left click is required to select drives and files. A beep occurs if the files are not of recognized format.
  • The central area displays image files selected in the file list. Moving over the displayed image with the right mouse button down, and then releasing it, will create a Cropping region. Dragging and dropping the display over to the box above the Assign button, with the left mouse button will copy and size the image or its cropped region for use as one of many Quote Thumbnails. These images are then assigned to a selected quote only when the Assign button is clicked.
  • At the bottom left is a listbox containing the entire collection of Quotes. These can be edited by selection and keying in the change in the input section at the top of the box.
  • An Assign button. Edit changes at the top of the listbox are lost if the Assign button is not activated before selecting another item in the listbox.
  • An Append button. When clicked a new quote "New" is created and inserted alphabetically in the list of Quotes.
  • A Delete button that can remove the selected quote and its associated image from the collection.
  • To the right of these is a scroll bar that determines the scroll offset of the quotes on the RHS of the screen. This value is displayed in the scroll counter, situated above the Assign button. Dragging the scroll slider changes the scroll counter faster. The purpose of the RHS screen is discussed in Thumbnail Indexing. If the Generic Quotes Database is loaded, left-click on any blue or orange values or indices to hear their sounds, or right-click to invoke the Bitmap Finder Screen.
  • A compact button on the bottom can be used for maintenance on the Database.
  • Selecting the square crops checkbox forces all rectangular Cropping regions to snap to square when the right mouse button is released.
  • When the "Write to Text" checkbox at the bottom left of the screen is ticked, there is a prompt for the text version of the Quotes Database to be written to a text file upon Quitting the screen.
  • Activating the Quit on the bottom left quits the screen.
Notes:
All images going in or out of the Database are written to temp disk files, so depending on number of open explorer windows and/or disk access speeds, a screen refresh/change might be take a few extra milliseconds. On older rigs Quotes Database modifications are significantly longer.
The Assign & Delete buttons have no corresponding undo, and modifications with them cannot be undone even by clicking "Cancel" back on the Profile Configuration Manager Screen.
The Quotes Database does not accept duplicate records.
If an item in the listbox has a Thumbnail assigned to it, then Deleting it or Assigning another thumbnail to it will produce a warning in the case other items have been indexed to the original Thumbnail.
