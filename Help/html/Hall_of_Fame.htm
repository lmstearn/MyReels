Hall_of_Fame

The Hall_of_Fame Database located in the file Hallfame.s$t and viewed on the Hall_of_Fame Screen, holds 15 statistical items of each game for which data has been written to this Hall_of_Fame Database. The Statistics are written to Slotdata.s$t for each session before the final commit to the Hall_of_Fame at game completion.The size of this Database can increase to many millions, but to exactly how many, who knows?
