The Appearance Tab of the General Options Screen features:

  • At top are 18 buttons for changing the colours of the various areas of the Game Screen. As an alternative to Background and Game Title area colour, bitmaps can be selected for tiling in with the radio controls. New bitmaps can then be located with the corresponding "Select" Button.
  • On the lower right is a section for changing the fonts used in the Game Screen.
  • A section for changing the Properties of the Active Candy Spin and Bet Multiplier buttons.
  • A list of options on flashing Lit Prize Amount and Picture Multiplier Labels when a win occurs on the Game Screen.
  • A check for using Big Pictures.
Notes:
If the Game Title area has the same directory and file name as the Background bitmap, a Tooltip on the Background buttons indicate the two areas will be merged.
Equivalent sizes of a range of fonts as seen in in font dialog boxes vary according to type of font and source of font derivation. Thus the font sizes in the select font dialog boxes are ignored when scaling the Game Fonts in the Game Screen.
