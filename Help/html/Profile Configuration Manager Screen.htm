The Profile Configuration Manager Screen consists of:

  • Pictures on the left, when clicked, get the General tab of the Picture Properties Screen.
  • Pictures on the upper right are visible when two or more Pictures are used in a Free Game or Free Spin Feature. Their vertical Ranking resolves any possible Free Game & Free Spin Contentions.
  • The Substituter / Naturals Bonus Multiplier Value Ranges.
  • The Bet Multiplier Spin Boxes for the Value Ranges.
  • Toggles sumnation of highest / every win on a Payline. If "Highest wins" is chosen, every Prize Amount of equal value on a Payline is paid, however.
  • Toggles Feature Multipliers, applicable when one or more Payline is activated.
  • The Random Jackpot starting amount and increment Spin Boxes.
  • The Money Back trigger and duration Value Ranges.
  • The New Game button and starting cash Value Range.
  • The Value Of Game, if applicable. It is displayed in the same format as the game Statistics on the About Screen, having a Tooltip indicating Probability Of No Prizes.
  • A "Save & Quit" button when clicked commits changes on all configuration Screens since the last time this button was activated.
  • A "Cancel" button when clicked disregards all changes made since the current load of the Profile Configuration Manager and quits to the Game Screen.
  • A "General Options" button when clicked gets the General tab of the General Options Screen.
  • An "Edit Reel Configuration" button when clicked gets the Edit Reels Configuration Screen.
  • A "Recalculate Return" button when clicked updates the Value Of Game.
  • A "New Reel Pictures" button when clicked gets the Configure New Pictures Screen.
