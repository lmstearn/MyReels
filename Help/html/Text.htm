The Text Tab of the General Options Screen features:

  • 8 boxes of text with self explanatory descriptions. No more than 20 characters allowed in the boxes of shorter length such as the Game Title, no more than 30 in longer boxes. Quality of display of these captions on the Game Screen will very much depend on the type of Game Fonts selected in the adjacent Appearance Tab.
  • A list of Prize ranges, each having a corresponding caption displayed in the Game Messages Area. 30 characters or less.
  • Raw text file selection for displaying Quotes: 100 characters or less required for the directory pathname. The section is not available when Big Pictures are used. If "Randomise" is unchecked, the Quotes will be shown in sequential order in a cyclic fashion. The Quotes file in the MyReels base or Installation directory is always used if the "Use Base Dir." box is checked.
  • The "Configure Thumbnails" button gets the Configure Thumbnails Screen. Can take some time to load if the Quotes Database is large.
