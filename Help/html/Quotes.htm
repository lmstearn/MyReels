Quotes

There is a facility on the Text Tab of the General Options Screen to create a Quotes Database with the option of Quote Thumbnails. The requirement is an ASCII coded file containing text separated by carriage returns.

Any lines of text longer than 172 characters, (longer text creates problems with display using different fonts) will be automatically truncated. No more than 32767 quotes allowed, according to scroll limitations in the Configure Thumbnails Screen. The Database stores records in alphabetic sequence based on the initial text of each quote.

Filesizes and available space appear as a Tooltip on the "Select Quotes file" button.

If there are issues with either the text file or Database a notification invariably pops up indicating cause and the Quotes generation will fail.

When a new Profile Configuration is created in a different directory to the install directory, a new Quotes.s$t is created, Filesizes and available disk space appear as a Tooltip on the "Open/Create a new Save Directory" and "Allocate Pictures to ...." buttons on the Configure New Pictures Screen.

The Quotes file bundled at Installation and offered as an optional component in the Updater uses a diverse range Sources on the WWW.
