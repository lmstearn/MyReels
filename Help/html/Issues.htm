Issues

  • Reels spin for too long! This is a result of how VB6 and post Vista systems interpret the handling of MHZ registry values. Solution: Find the PC Speed dropdown list on the right of the General tab of the General Options Screen and select a lower value.
  • Some Screens in this game don't readjust well if either the Windows Taskbar is resized, or not situated at the bottom of the Screen.
  • For users of Coolsoft's VirtualMIDISynth: After a fresh reboot, the first application that assigns midi ports will "own" the ports, excluding other apps which may then fault when accessing them. If MyReels "owns" the port, selecting different Midi ports does not vary the output according to the arrangement in the VirtualMidiSynth Configuration Soundfonts Tab, returning only that of the Default Midi Out Device configured in the Midi Mapper Tab. Partial Workaround: Change the Default Midi Out Device in the VirtualMidiSynth Configuration. Refer to this thread for further developments.
  • MyReels might only recognize the first portion of the Quotes Database in the base directory. Solution: Toggle Use Base Dir. check.
  • On an XP machine an occasional C0000005 segment fault in NTDLL.DLL was also experienced when quitting the game without closing this help window first. This was also related to Midi issues and addressed by code refactoring and should not reoccur.
  • A missing "nn.bmp" occurs on opening a configuration that was created previously by using the "Alllocate" button from the Configure New Pictures Screen in a particular combination of actions. See OpenError for a workaround.
  • Certain external pages can produce errors, take a long time to load or not render properly in HTMLHelp. A partial registry workaround is available at the (slow loading) Homepage.
  • Machines running IE without CSS3 may not get the stretched background bitmap in this help file. Depending on graphics modes on such machines (e.g. 256 colors), the fade on the background is not noticeable.
  • Using the mouse to Crop large pictures in the Configure New Pictures Screen or Configure Thumbnails Screen may cause the scrollbars to "race" up/down or left/right.
  • Errors when launching MyReels for the first time while the installer is active- but not when launched from the Installer. Solution: Complete the install process first.
  • Limited multi-lingual support for the game, help and installer.
  • If a Profile Configuration folder is manually moved to a folder on another machine, the string names for the sound and music files in Slotdata,s$t will only update to the new Installation directory if they are identical to the Generic names, else they will have to be individually modified to work. The Background bitmap file is handled differently.
  • Winning Scatter combinations in a Multiline
    • game are not multiplied by the number of Paylines. More infornation at Github.
  • MyReels 1.X, 2.X: In certain scenarios the old Wise Installer will not write to Software\Microsoft\Windows\CurrentVersion\Uninstall\MyReels even with admin privileges. Thus uninstallation must happen from the MyReels application base directory.

    • Note: Some issues of display can be resolved using the Graphics Debug page.
