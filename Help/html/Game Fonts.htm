Game Fonts

MyReels does not calibrate any custom font pitch or size for rendering according to font name or type in the Game Screen. It's a one size fits all approach based on a scaling factor variable in the code: textwidthratio, usually calculated as being the smallest or next to smallest size of the font. If, for example, the smallest size of a custom font is 16, the text is likely to either go off control's display area or to clip within the control. High Resolutions can also cause the display of text with certain fonts to clip.

For suscribers to Vista, and onwards, MyReels is DPI aware so there should be no issues when changing Windows Display Scaling (Scale & Layout: text, apps other items) in the Control Panel to more than 100%.
