The Hall_of_Fame Screen features:

  • At top, a scale representing degrees of success of the displayed past glories.
  • A Coat Of Arms.
  • Below, the Hall_of_Fame table initially showing items 1_to_8:
  1. Player Name.
  2. Game Title.
  3. Startcash.
  4. Endcash.
  5. Reason code for game termination:
    • 1 if Money On Hand is zero.
    • 2 if at least one of the Game Statistics has reached or exceeded the Game Limit.
    • 3 if game was terminated by starting a New Game.
  6. Date of completion.
  7. Grand total of Spins in the game.
  8. Number of times the values in the Edit Reels Configuration Screen have been modified.
  9. Number of modifications of Picture Properties except Free Games, Free Spins and Substituters.
  10. Number of modifications in the Profile Configuration Manager Screen.
  11. Number of times Statistics have been reset from the About Screen.
  12. Number of times the Game was played with the same Random Seed value.
  13. Sampled Value of Game.
  14. Last recorded VOG or Value Of Game. Cell is orange when VOG changes are saved during the course of the game.
  15. Number of Monte Carlo simulations used to estimate the Value Of Game.
  • A horizontal scroll bar to show the 8_to_15 group of above Statistics
  • An Ok button to return to the About Screen.
Notes:
Any row in the table containing 12 blue cells is, of course, the best result, and guarantees candidacy to the MyReels_Honours_List appearing "shortly" in the up and coming MyReels.com site.
An installed Courier New font is required to view this screen.
