Scatter

A Scatter Picture can occur above or below the middle Payline to qualify for a Prize Amount conforming to a given Pay Method. For considerations of symmetry the Prize Amount ratios for the Prize Levels are fixed. There are no single Prize Amounts, Middle Threes, or Substituter options available for Scatters. Either 1, 2 or 4 Scatters can appear on any of the 5 Reels but this number is the same on all other Reels. For convenience, Scatter Prize Amounts are shown towards the bottom of the Game Screen.

Scatters are configured by selecting Pictures in the Edit Reels Configuration Screen.

A maximum of two Pictures, only, are allowed to have Scatter status in a game.

If the highest win paid only option in the Profile Configuration Manager Screen is selected, Scatter wins will be matched against middle Payline wins only.

As Scatter Prize Amounts are not multiplied by the number of Paylines bet, there are better chances in a single Payline game using Scatters.
