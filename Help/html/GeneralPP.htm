The General Tab of the Picture Properties Screen features:

  • A Scatter checkbox. Once the Scatters are generated from the Edit Reels Configuration Screen, they can be deselected and reselected here.
  • The Pay Method options.
  • Prize Amounts Value Range Spin Boxes.
  • A list of all the Pictures in the game available as potential Substituted Pictures. Clicking on any of these will either make the selected (large) Picture a Substituter with a blue dot or display a Tooltip explaining why not.
  • The list of Reels for which each the Substituter can be chosen as active or inactive.
  • An Accept button: all changes are actually committed or rejected from the Profile Configuration Manager Screen.
