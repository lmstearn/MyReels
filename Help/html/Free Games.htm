The Free Games Tab of the Picture Properties Screen features:

  • At top a check to activate the Free Game Feature
  • A choice of patterns to trigger the Feature. For a description of patterns, see Pay Method.
  • A Feature Multiplier and No Win Bonus Value Range Spin Boxes.
  • Settings for the number of Spins in the Feature.
  • A Value Range specifying how many new Feature combinations during the current Free Game are Stored. If this is unchecked then this current Free Game is always replaced by a new Feature, whenever such a combination actually turns up, see Feature-Reset.
  • A check for Naturals only, to trigger a Free Game.
Note:
The given choices of Free Games in the Tab range over Reels containing at least one of the selected Picture, regardless of number and distribution of possible Substituters. If a Substituter is used, this restricted choice doesn't prevent Feature triggers on Reels where our Substituted Picture doesn't exist.
