Version History

Version 1.2:
Excessive Spin Button depression causing Slotdata.s$t corruption fixed. Bet total now adjusted for Features. Fixed Feature Prize Amount labels zeroing prematurely. Premature Money Back at beginning of Features fixed. Some menu commands disabled during Features. Improved Help File. Run time error 401 on "Game Limit Reached" fixed. Free Games now available for more than 3 of the same qualifying Pictures on Payline. Old Quotes Database not modified and the new Quotes Database compacted when creating a new Profile Configuration.
Version 2:
Bug fixes: Fixed various anomalies and clarified matters relating to Random Seed assignment of Reels arrangements. Fixed problems relating to restoring current Defaults. Fixed problem of Naturals Game Limit reached in premature End_of_Game. Overhauled both the MyReels Value Of Game routine and the summation of Substituter Prize Amounts. Fixed nopay on some PXPPP combinations. Fixed bug of not closing midiport properly (whoops). Accelerated the spin controls of the Prize Amounts. Fixed various Bonus Multiplier bugs. Input error 21 bug fixed. Fixed font size nag in setting game and Game Title fonts. Fixed minor display bugs in the Edit Reels Configuration Screen. Allowed greater pause times between Spins in Features.
Version 2: (cont.)
Changes made to Hall_of_Fame with addition of "Stats Resets" and "Monte Carlo". Added the new Monte Carlo feature in the General Tab of the General Options Screen, removing the Compact Hall_of_Fame button. Added Multiline Statistics to the About Screen, removing the "Stats Resets" button to the bottom of the Screen. Added a Multiline feature on the Game Screen and Tooltipped the prizemeter with a breakdown of Multiline Prize Amounts won.. The Screens will now stretch for 1024 X 768 Resolution. Some Quotes added.
Version 2.01:
Fixed various problems of pays when the Highest win option in the Profile Configuration Manager Screen is selected. Display of the prizemeter fixed, Multiline wins fixed.
Versions 2.02, 2.03:
Fixed various problems relating to Version 2.X additions & bug fixes.
