The General Tab of the General Options Screen features:

  • At top a check that when toggling through the Bet Multiplier values on the Game Screen, the values used in the game's Random Sequence will now be taken from a different point within that Sequence.
  • A check if the game restarts (recommences at the start of its Random Sequence) after leaving the Profile Configuration Manager Screen. If this is unchecked, then the game continues in the current Random Sequence. Whether checked or unchecked, there is no effect when the Random Seed itself is changed (see below) between configuring and loading the Game Screen.
  • A list of options on the time frame for a particular Random Sequence to be preserved and button for obtaining a new Random Seed.
  • Four drop-down lists of parameters determining the manner and outcome of the Spins.
  • A list of PC speed parameters determining Picture animation in the Game Screen.
  • A choice of saved, current or Generic Profiles that can be used as a Default when starting a New Game or a new Profile Configuration. On the right are buttons to save current Profiles, and restore saved and/or Generic Profiles.
  • A list of options on the Game Limit.
  • A list of numbers relating to the Monte Carlo simulator. Note: This is available for change only at the start of a New Game.
  • A check on showing the About Screen on exit.
  • A facility to adjust for changes in the location of the Hall Of Fame Database, (100 characters or less) required for the directory pathname.
