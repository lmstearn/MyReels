Filesizes

When a new Profile Configuration is created in a different directory to the install directory, a new Slotdata.s$t is created, Filesizes of the newly created files and available disk space are shown as Tooltips on the "Open/Create a new Save Directory" and "Allocate Pictures to ...." buttons on the Configure New Pictures Screen.

Filesizes for files to be created and available disk space also appear as a Tooltip on the "Select Quotes file" button on the Text Tab of the General Options Screen.
