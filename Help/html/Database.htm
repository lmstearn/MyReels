Database

The program uses 3 disparate Microsoft Jet (tm) 2.X compatible databases:

  • Slotdata.s$t contains all input and Profile Configuration data and is critical for the game to run. When a new Profile Configuration is created in a different directory to the install directory, a new Slotdata.s$t is created in that directory.
  • Hallfame.s$t is a non critical Hall Of Fame database file and it is possible to have more than the recommended one copy on a drive accessible to your device.
  • Quotes.s$t is a non-critical Quotes database file. When a new Profile Configuration is created in a different directory to the install directory, a new Quotes.s$t is created in that directory.
Notes:
All databases are compacted as a function of usage, but large Quotes files benefit from manual intervention.
The Database DLL this program used under construction was DAO350.DLL. There are newer versions available in the form DA03**.DLL, and shouldn't adversely affect the Installation or performance of this program.
