The Edit Reels Configuration Screen features:

  • At top, a button labelled "Regenerate All". This will generate a fairly random set of numbers, replacing the entire range shown on Screen, ordering them according to Rank as designated in the checkbox below. It will also reset any Scatter allocations.
  • A check for the displayed numbers to be arranged in ascending order, the smallest corresponding to the Picture of highest Rank.
  • Pictures, when clicked will be, under specific conditions, selected as Scatters.
  • Associated with each Picture is a row of 5 numbers, corresponding to the 5 Reels. Each number represents the numerical occurrence of the Picture on the specified Reel. Each of the 2 sets of 5 columns of numbers (the RH columns refer to the second set of 7 Pictures) must add to 24, or Reels generation will not occur. Right clicking a number will decrement its value by one, left clicking will increment its value by one. Restrictions apply if the "Ordered" option is chosen. Up to four of these numbers can be zero, thus excluding a Picture from a Reel altogether. Any arrangement with at least one excluded Picture is termed an Ordinatio Partialis.
  • Buttons with "Reel" labels that perform the same function as "Regenerate All," except the generation only applies to their corresponding column of numbers. They are not available if at least one of the Pictures is a Scatter.
  • A "Generate" button at centre top quits to the Profile Configuration Manager Screen and commits the current Picture allocations to a new random arrangement on the Reels. All Substituters, Free Games and Free Spins are nullified or reset when this occurs.
  • Activating the "Cancel" button will discard any changes made on this screen and quits to the Profile Configuration Manager Screen.
