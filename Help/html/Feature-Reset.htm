Feature-Reset

A Feature-Reset occurs when a new Free Game or Free Spin Feature replaces an old one before the maximum number of Spins in the Feature has completed. This takes effect as soon as the combination is detected, so that other similar possible Feature candidates on other Paylines may not be used or Stored.

In a Multiline game, the middle Payline is always considered first, the bottom Payline, when active, is always considered last. Scatter Features are treated as middle Payline combinations.
