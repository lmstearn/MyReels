Resolution

MyReels 3.X is DPI Aware and adjusts up to and including 1080p. In resolutions equipped with greater PPI, some controls, particularly the Spin Box & Candy buttons may not show correctly, so a program like PrgLnch is recommended.

For multi-monitor setups, again a program like PrgLnch can launch MyReels in a lower resolution, but perhaps not in a Primary monitor, as being a VB6 compilation MyReels can only use the Primary Display settings in a multi-monitor environment.

Choosing the Big Pictures option in any higher resolution than 800X600 won't improve the quality per-pixel of a rendered Picture in-game.

Please Quit the game before changing the Resolution or other settings of your Windows Display.

Notes for MyReels Versions 1.X & 2.X:
This game was originally designed for a Screen Resolution of 800 X 600 pixels and will scale up to resolutions such as (the tested) 1024 X 768, 1152 X 864, 1280 X 1024 or wide screen 1280 X 720, 1366 X 768, 1600 X 900.
Scaling issues may depend on the graphics card: for instance, in 1280 X 1024, the Game Screen ran off the bottom edge of the CRT monitor on a certain XP machine sporting an S3 graphics chip.
For Pre-Vista Users, setting Text/Fonts to anything exceeding 100% of "normal" size through Control Panel/Windows Display Properties/Settings may present issues with the display of the Game Fonts.
