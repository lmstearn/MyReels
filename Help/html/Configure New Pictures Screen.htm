The Configure New Pictures Screen features:

  • On the top left, a drive list, directory list, file list and a Picture file filter. A double click is required to change directories, a single left click is required to select drives and files. A beep occurs if the files are not of recognized format.
  • The central area displays image files selected in the file list. Moving over the displayed image with the right mouse button down, and then releasing it, will create a Cropping region. Dragging and dropping the display over to one of the boxes on the right hand side of the Screen with the left mouse button will copy and size the image (or its cropped region) for use in the game.
  • On the RHS of the screen are boxes reserved for images just selected representing the new Pictures. Allocated images will be ordered in Rank. Images in the boxes can be replaced by other images and an image allocation can be cancelled altogether by clicking in the box.
  • On the bottom area is the "Open & Create a new Save Directory". Activating this will provide the home for the new Profile Configuration, a location which should be set before clicking the "Allocate ..." button below. After clicking the button, the Player must choose a working directory (typically the MyReels Base Directory) where new subdirectories can be created if desired. This button and the one below it have Tooltips regarding Filesizes and available diskspace. Note: In this Version, directory names exceeding 100 characters aren't allowed.
  • Activating the "Allocate ..." button commits the new images, deletes old files used by the game ("nn.bmp"), if they exist, and copies the currently used Slotdata.s$t and ("nn.bmp") all to the new home directory.
  • A "Next" button when clicked copies the Quotes settings to the new directory and proceeds to the Edit Reels Configuration Screen.
  • A checkbox labelled "Square crops". Toggled on forces all rectangular Cropping regions to snap to square when the right mouse button is released.
  • A check for the creation of a Shortcut when the Next button is clicked.
Notes on the "Next" Button:
As Slotdata.s$t is updated with new Picture data only when the "Next" button is clicked, just the last of the directories you have created or allocated on this screen will contain the new Profile Configuration data that is stored in Slotdata.s$t. Note: A particular combination of actions on this screen can lead to an issue on a subsequent game load- also see OpenError.
A New Game commences each time a new Profile Configuration is created. Current data is also automatically saved when a new directory is either accessed or created.
