Monte Carlo

A number specified in the General Tab of the General Options Screen determines the number of simulated games required to provide an estimate of the Value Of Game. The larger the number is, the better the estimate will be. If the number is 0, then the Monte Carlo option will not be used and MyReels will do its best to provide an alternative estimate.

If large numbers are chosen, calculating the Value Of Game can take a long time, especially if lots of Substituters and Features are used in the Profile Configuration. However the Estimate of Calculation times is continually decreasing as processors increase in grunt power.

During the Calculate processing time MyReels will become unresponsive, with an hourglass (or equivalent) mouse cursor, until all iterations are complete.

If large Monte Carlo values are chosen, it makes sense to Recalculate Return after all Profile Configuration changes, so that it needs only to be done once before returning to the Game Screen.

Note: A Monte Carlo value cannot be changed without starting a New Game.
