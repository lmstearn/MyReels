The Sounds Tab of the General Options Screen features:

Configuration dialogs on the Sounds & Music of MyReels:

  • At top left a check to enable Sounds
  • At top left a check to enable sounds for the Quote Thumbnails. It's not available when using Big Pictures.
  • A section for wave files with a list of event descriptions and a corresponding list of wave file names. Wave files are first selected from the RH list of wave file names, and can then be either chosen with the "Browse" button (100 characters or less required for the directory pathname and filename), or from the list of System Sounds below. Once selceted, they will appear in the text box to the left of the "Test" button at top. On being satisfied the selected Sound fits the event description, only the "Add" button will commit the new change. The selected Sound can also be removed with the "Clear" button.
  • A check to display only currently assigned System Sounds.
  • The Midi files section also contains a list of event descriptions and a corresponding list of Midi file names. If the "Randomise" is unchecked, the background files will be played in the listed order in a cyclic fashion. If "Randomise" is checked, a new random selection is generated when a Spin is initiated through the Game Screen. If the game is left idle, however, the playlist order reverts to cyclic, regardless of this setting. Midi file names selected with the "Browse" button are required to be less than 101 characters in length.
  • A list displaying enumerated Midi devices. Prior to XP, the selection of 0 prevented MyReels from using Midi altogether, where it could only be re-enabled here, not from the Game Screen. In newer versions of Windows it is equivalent to turning the music off.
