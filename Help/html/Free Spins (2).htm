The Free Spins (2) Tab of the Picture Properties Screen features:

  • At top a check if the Feature can be triggered when the held Pictures pay a Prize Amount.
  • A check for Naturals only, to trigger a Free Spin.
  • A Value Range specifying how many new Feature combinations during the current Free Game are Stored. If this is unchecked then this current Free Spin is always replaced by a new Feature whenever such a combination actually turns up, see Feature-Reset.
  • A Feature Multiplier and No Win Bonus Value Range Spin Boxes.
  • Settings for the number of Spins in the Feature.
