Sampled Value of Game

The Sampled Value of Game or SVOG is just the total amount the Player has won divided by the total amount the Player has bet to date, expressed as a percentage. If there has been a large number of Spins, this value should approximate the Value Of Game.

An adjusted SVOG factor seen in the About Screen in any of the Prize Categories represents the total amount won in all Prize Categories divided by the total amount won in the Category.
