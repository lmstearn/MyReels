ActiveX Maintenance

A useful tool for viewing the CLSID/ProgID registrations for dll/ocx files is NirSoft's RegDllView. For cleanup:

Windows Systems up to and including Win2k:
This can be unregistered by clicking the Start button ,selecting Run, typing regsvr32.exe /u ACTCNDY4.ocx and clicking OK, before manual deletion from the &winsys& directory. Many NT based machines have Steve Miller's Dependency Walker - which can be otherwise obtained through a web search engine. Faber Toys is another very useful example.
Windows Systems XP & Vista
Use Windows disk cleanup or try CCleaner.
Windows Systems 7 and Onward:
Use Windows disk cleanup.
