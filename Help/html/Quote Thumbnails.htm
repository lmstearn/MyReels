Quote Thumbnails

Versions 2.2 and later use an extended Database file containing 17.4k thumbnails associated with the Generic collection of Quotes included at Installation. These are set in the Configure Thumbnails Screen, accessible from the Text tab of the General Options screen.

Version 2.2.4 and later has sounds associated with the 440+ thumbnails in the Generic Profile Configuration which are played when either Spinning or by clicking the thumbnail in the Game Screen. The sound data currently coded in MyReels.exe is planned for addition to Quotes.s$t at some later date.
