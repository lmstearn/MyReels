New Game

A New Game can be started either by setting up a new Profile Configuration from the Configure New Pictures Screen or restarting the same Profile Configuration in the Profile Configuration Manager Screen.

In the first case, all Picture Properties and Scatter Properties are reset. If an old Slotdata.s$t is detected in the directory just selected for the new Profile Configuration, the Hall_of_Fame directory name specified in the old Profile Configuration will be the one used for writing old Statistics to, before it is replaced by the Hall_of_Fame directory name in the current Profile Configuration. If Slotdata.s$t is not detected, the current Slotdata.s$t is copied to the new directory.

In both cases, all relevant Statistics are written to the Hall Of Fame, all Pots are reset and the Money On Hand reverts to the starting value specified in the Profile Configuration Manager Screen. Recalculating Return should be the last activity before quitting to the Game Screen.

Starting a New Game uses settings in the selected Profile Configuration Default.

A New Game automatically starts after program Installation.
