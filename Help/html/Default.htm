Default

A subset of available Profile Configuration values can be used as New Game Defaults and is set through the General Tab in the General Options Screen. They do not include Reels distributions, Pictures or anything set in the Picture Properties Screens.

One subset is a saved group of values. This is not available if no defaults have been saved,

Another subset is the in-built Generic Default which cannot be directly modified.

Finally there is a current or working subset of Defaults which applies directly to the most recent load of the Profile Configuration including any modifications made since.

Note: As just one Monte Carlo value applies for each game played, it can be restored from Defaults only on starting, or, immediately after starting, a New Game.
