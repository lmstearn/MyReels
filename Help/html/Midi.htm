Midi

The music of MyReels comes from midi playback via Windows Media function calls.

If Coolsoft's VirtualMIDISynth along with the attendant CoolSoft MidiMapper are not installed, only the default Roland GS Wavetable Sound fonts are used. The selected port N in the Sounds Tab of the General Options Screen should correspond to the VirtualMIDSynth #N device, - for more details refer to this thread. For improved stability also check "Enable hardware mixing" in the VirtualMidiSynth Options Tab.

The following only applies to Systems prior to

Windows XP

:

The MCI calls for the Midi interface do not take very kindly to file or directory names longer than 8 characters. This has been partially addressed with point-blank truncation, but outstanding issues remain in the case of:

  • a) 2 or more file names in the same directory each containing more than 8 characters but the first six characters of each name are identical.
  • b) 2 directory names branching off the same parent directory each containing more than 8 characters but the first six characters of each name are identical.

The first file in case a) or first directory in case b) that appears in alphabetical order will always be chosen irrespective of the selections made on the Sounds Tab of the General Options Screen.

Using shorter .mid and .wav file & directory names then might solve problems with their playback.
